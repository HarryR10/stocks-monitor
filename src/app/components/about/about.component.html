stock.tools

- is a simple application for displaying stock quotes.

In this app you can:

  • find stock symbols of companies;
  • get information about stock quotes as a .csv file;
  • display quotes charts;
  • display news for the selected company.

The app uses some public api to get financial market data. For the stable operation of this application, it is enough to have free public api keys. You can get them on the following resources:

Searching in app:
Alpha Vantage API
Extract and display data:
iex cloud API
